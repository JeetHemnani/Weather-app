
This is a simple weather application built using Flutter. I created this project while learning the basics of Flutter, focusing mainly on frontend development and API integration. The app uses the OpenWeather API to display real-time weather data for different cities.

Features
	•	Simple and clean user interface built with Flutter
	•	Real-time weather information
	•	Search weather by city name
	•	Displays temperature, weather conditions, and icons
	•	Weather data fetched from the OpenWeather API

Tech Stack
	•	Flutter for frontend development
	•	Dart programming language
	•	OpenWeather API for real-time weather data
	•	HTTP package for making API requests


API Used
OpenWeather API
https://openweathermap.org/

Learning Outcome
	•	Learned how to design basic UI in Flutter
	•	Understood how to integrate APIs in Flutter
	•	Worked with JSON data from an external API
	•	Learned how to handle real-time data in an app

How to Run
flutter pub get
flutter run
